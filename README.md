# 🍜 Hostel Quick Meal Finder

A web app that helps hostel students find meals using ingredients they already have.

## Features
- Search meals by ingredient
- Sort meals A-Z and Z-A
- Filter Vegetarian / Non-Vegetarian meals
- Save favorite meals
- Remove favorite meals
- View recipe instructions
- Dark / Light mode
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
- TheMealDB API

## How to Run
1. Download the project
2. Open `index.html` in browser

## Deployment
The project is deployed using GitHub Pages.
